Автомобильный Сортировщик

Описание проекта

Данный проект реализует алгоритмы сортировки для коллекции автомобилей с полями: модель, мощность, год выпуска.
Проект демонстрирует применение паттернов проектирования, функционального программирования и многопоточности в Java.

Основные функции проекта

1. Класс Car
- Модель (String)
- Мощность (int) 
- Год выпуска (int)
- Реализация паттерна Builder

2. Паттерн Стратегия
- Реализации сортировки:
  - ByModel (по модели)
  - ByPower (по мощности)
  - ByYear (по году)

3. Дополнительные функции
- Сортировка с четными/нечетными значениями (дополнительное задание 1)
- Запись отсортированных коллекций в файл (дополнительное задание 2)
- Заполнение коллекций с использованием стримов (дополнительное задание 3)
- Многопоточный подсчет вхождений элементов (дополнительное задание 4)

Для тестирования используется подход "ручных" тестов без использования внешних библиотек тестирования.
